📋 ขั้นตอนง่ายๆ:

1. ติดตั้ง gcloud CLI (ถ้ายังไม่มี)
   → https://cloud.google.com/sdk/docs/install

2. Login และสร้าง Project
   → gcloud auth login
   → gcloud projects create riffai-demo

3. รันสคริปต์ Deploy
   → cd riffai-platform
   → deploy-gcp.bat        (Windows)
   → ./deploy-gcp.sh       (Linux/Mac)

⏱️  ใช้เวลา: 10-15 นาที
💰 ค่าใช้จ่าย: ฟรี (ถ้าใช้น้อย)

═══════════════════════════════════════════════════════════════

📚 คู่มือทั้งหมด:

✅ START-DEPLOY.md
   → เริ่มต้นง่ายๆ ทีละขั้นตอน

✅ DEPLOY-README.md
   → เปรียบเทียบวิธี deploy 3 แบบ

🎯 แนะนำ:

• ทดสอบ/Demo → ใช้ deploy-gcp.bat (ฟรี, SQLite)

• Production → ดู DEPLOYMENT-CHECKLIST.md (ครบทุกอย่าง)

✨ ไฟล์ที่เตรียมไว้แล้ว:
